Explicación de etiquetas HTML

Este documento responde: ¿Para qué sirven DOCTYPE, html, head, meta, title y body?

<!DOCTYPE html>

Qué es: Una declaración (no es una etiqueta que cierre) que aparece la primera línea de un documento HTML.

Para qué sirve: Indica al navegador que la página usa HTML5. Ayuda al navegador a renderizar correctamente el documento.

<!DOCTYPE html>

<html>

Qué es: El elemento raíz que contiene todo el documento HTML.

Para qué sirve: Encierra <head> y <body>. Se le suele poner el atributo lang="es" para indicar el idioma (útil para lectores de pantalla y buscadores).

<html lang="es"> ... </html>

<head>

Qué es: La sección del documento que contiene metadatos y recursos.

Para qué sirve: Aquí se coloca información que no se ve directamente en la página: título, meta tags, enlaces a CSS, favicon, scripts que se cargan antes del contenido, etc.

<head> (meta, title, links a css, scripts) </head>

<meta>

Qué es: Una etiqueta para definir metadatos concretos (es vacía, se escribe sin cierre).

Para qué sirve: Define el juego de caracteres (charset), cómo se adapta la página en móviles (viewport), descripción, autor, y muchas otras cosas que ayudan al navegador y a buscadores.

<meta charset="utf-8">
<meta name="viewport" content="width=device-width, initial-scale=1">

<title>

Qué es: El título del documento (solo uno por página).

Para qué sirve: Aparece en la pestaña del navegador y en los resultados de búsqueda; ayuda a identificar la página.

<title>Mi título de página</title>

<body>

Qué es: La parte visible del documento HTML.

Para qué sirve: Contiene todo lo que el usuario ve en pantalla: textos, imágenes, formularios, enlaces, tablas, etc.

<body>Contenido visible de la página</body>
